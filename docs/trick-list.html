1. 3 2 1 Bum

Wymagania: Mechanika 3+, Materiały wybuchowe 3+

Mechanika: Uzbraja pojazdy/urządzenia napędzane paliwem w bombę zegarową.

Narracyjne: Efekt oznacznik — brak automatycznej mechaniki. Utwórz pasywny AE jako przypomnienie dla MG/gracza.

2. Aramis

Wymagania: Zręczność 14+, Broń ręczna 5+

Mechanika: Akcja za 2 sukcesy: wytrąca broń przeciwnika. PT testu o poziom wyższy.

getMeleeActions Script Akcji
args.actions.push({
  id: "aramis-disarm",
  name: "Aramis – Rozbrojenie",
  successCost: 2,
  minDice: 2,
  maxDice: 2,
  damage: "—",
  annotation: "Aramis: broń przeciwnika leci na bok."
});
dialog Hide Script Ukryj w dialogu
return args.rollType !== "melee";
dialog Dialog Script Skrypt dialogu
args.fields.difficultyShift += 1;

6. Berserker

Wymagania: Odporność na ból 6+, Budowa 12+

Mechanika: Po otrzymaniu Ciężkiej rany w walce wręcz: auto-sukces wszystkich testów Odporności na ból.

immediate Aktywacja efektu
await this.sendToChat("<b>Berserker!</b> Stan aktywny — automatyczne sukcesy Odporności na ból.");
await this.actor.setFlag("neuroshima", "berserker", true);
preRollTest Skrypt testu
if (args.test?.context?.skillKey !== "painResistance") return;
if (!this.actor.getFlag("neuroshima", "berserker")) return;
args.test.preData.autoSuccess = true;

21. Dłuższy strzał

Wymagania: Łuk 9+, Budowa 12+

Mechanika: Strzela z łuku na dwukrotnie większą odległość przy tych samych karach.

dialog Hide Script Ukryj jeśli nie strzelasz z łuku
return args.rollType !== "ranged" || args.weapon?.system?.rangedSubtype !== "bow";
dialog Dialog Script Połowa kary za dystans
// args.currentDistanceModifier = aktualna kara za odległość (tylko do odczytu)
// args.fields.distanceModifier  = delta — wartość odjęta od aktualnej kary za odległość
const dist = args.currentDistanceModifier ?? 0;
if (dist > 0) args.fields.distanceModifier -= Math.floor(dist / 2);
