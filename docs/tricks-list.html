Legenda Typów Implementacji

dialog

Modyfikuje parametry testu (modyfikator, poziom trudności, bonusy) przed otwarciem okna dialogowego rzutu.

preRollTest

Wywoływany tuż przed rzutem kośćmi. Umożliwia automatyczny sukces lub modyfikację redukcji kości.

takeDamage

Wywoływany przy otrzymywaniu obrażeń przez postać obrońcy. Ignoruje lub automatycznie zdaje testy ran.

manual

Wymaga aktywnego wywołania testu lub efektu przez gracza/MG za pomocą skryptu makra lub przycisku.

narracyjna

Sztuczki czysto opisowe lub fabularne. Niemożliwe do pełnej automatyzacji kodem.

pasywna

Działa jako znacznik efektu pasywnego informujący MG o braku konieczności wykonywania testów.

preTakeDamage

Wywoływany na OBROŃCY przed redukcją pancerza. Modyfikuje typ obrażeń (args.damageType) lub przebicie (args.piercing).

applyDamage

Wywoływany na ATAKUJĄCYM po wyborze typu obrażeń, przed pancerzem obrońcy. Używaj args.attackData.damageOverride do nadpisania tieru.

6. Berserker manual
Wymagania: Odporność na ból 6+, Budowa 12+

Składa się z dwóch efektów: pierwszy aktywuje stan Berserkera (manual), drugi (takeDamage) automatycznie zdaje testy odporności na ból i wyłącza się po zakończeniu walki (updateCombat).

Efekt 1 — Skrypt aktywacji (manual trigger)

// Sprawdź, czy aktor ma co najmniej jedną ciężką ranę (wymaganie)
if (!this.hasWounds(["C", "sC", "K", "sK"])) {
  this.notification("Berserker wymaga co najmniej jednej Ciężkiej rany!", "warn");
  return;
}
if (this.hasEffect("Berserker (Stan)")) return;
const eff = this.item.effects.contents[1].convertToApplied({
  "duration.combat": game.combat?.id
});
await this.actor.createEmbeddedDocuments("ActiveEffect", [eff]);

Efekt 2 — Skrypt rany (takeDamage trigger)

// Automatycznie zdaje testy odporności na ból podczas berserku
for (const wound of args.wounds) {
  wound.forcePassed = true; // auto-sukces testu odporności na ból
}

Efekt 2 — Skrypt końca walki (updateCombat trigger)

if (!game.combat?.started) {
  await this.effect.delete();
}

29. Garda dialog + preTakeDamage
Wymagania: Bijatyka 4+, Budowa 12+

Dwa skrypty na tym samym efekcie: dialog modyfikuje obronę i ustawia flagę aktora; preTakeDamage redukuje nadchodzące obrażenia.

Hide Script

return args.rollType !== "melee";

Dialog Script

// +6 do Zręczności w obronie
args.fields.attributeBonus += 6;

Submission Script

// Przechowuje flagę na aktorze do użycia w preTakeDamage
await this.actor.setFlag("neuroshima", "gardaActive", true);

Skrypt (preTakeDamage trigger — drugi skrypt na tym samym efekcie)

// preTakeDamage — redukuje typ obrażeń PRZED pancerzem przy aktywnej gardzie
if (!this.actor.getFlag("neuroshima", "gardaActive")) return;
if (!args.isMelee) return;
await this.actor.unsetFlag("neuroshima", "gardaActive");
args.damageType = this.reduceDamageType(args.damageType);
